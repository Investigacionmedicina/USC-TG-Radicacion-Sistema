# USC - Sistema de Radicación de Trabajos de Grado 2026

Sistema web de radicación para estudiantes de Medicina de la Universidad Santiago de Cali.

## 🚀 Características

✅ **3 Fases completas**: Solicitud Tutor, Aval Protocolo, Sustentación
✅ **Radicados únicos**: Identificador único para cada radicación
✅ **Fase 1 mensual**: Abre del 1-5 de cada mes
✅ **Panel Coordinadora**: Control completo y filtros avanzados
✅ **Descarga Excel**: Exporta todas las radicaciones
✅ **Carga de documentos**: Desde la plataforma
✅ **Correos automáticos**: Notificaciones a estudiantes y coordinadora
✅ **100% Gratuito**: Vercel + SharePoint
